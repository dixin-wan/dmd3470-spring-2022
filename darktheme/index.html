Dark Theme Potentials

Dark mode has gained a lot of traction recently. Like Apple, for instance, has added dark mode to its iOS and MacOS operating systems. Windows and Google have done the same. For example, The idea behind dark mode is that it reduces the light emitted by device screens while maintaining the minimum color contrast ratios required for readability. Both iPhones and Android handsets offer system-wide dark modes.
